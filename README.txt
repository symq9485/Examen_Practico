Desarrollar una pagina web que permita calcular el área de un triangulo, circulo
y rectángulo.

Al posicionar el puntero sobre el botón el fondo del body debe cambiar de color
según el área de la figura geométrica.

Una vez que el usuario haga click sobre el botón deberá mostrar el resultado del
área calculada atravez de una alerta. En caso de no haber ingresado previamente
los datos este debera notificarlo.
